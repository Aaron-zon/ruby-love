# 02 · MVP 功能范围

---

## MVP 核心目标

用最少的功能，做出**最完整的展览体验**。

---

## P0 · 必做功能

### P0-1 首页 · 展览入口

**功能描述**
- 全屏封面图 + 摄影师名字 / 网站标语
- 进入展览的入口按钮
- 简洁、有质感，第一眼就能定调

**验收标准**
- [x] 视觉冲击力强，不像普通网站首页
- [x] 明确告知观众"这是一个摄影展"
- [x] 入口按钮位置自然，引导流畅

---

### P0-2 作品集导航 · 系列分区

**功能描述**
- 将照片按主题 / 时间 / 地点分为若干系列
- 每个系列有独立封面、标题、简介
- 观众可以选择进入某个系列

**验收标准**
- [x] 系列数量 3～6 个（MVP 阶段）
- [x] 每个系列封面图能代表该系列风格
- [x] 系列卡片布局有设计感，不是简单列表

---

### P0-3 系列展览页 · 沉浸式浏览

**功能描述**
- 进入某个系列后，全屏展示该系列照片
- 照片之间有平滑过渡动效
- 每张照片可显示简短文字说明（可选展示）

**验收标准**
- [x] 照片以大图形式展示，不被裁切关键部分
- [x] 切换照片时有过渡动画，不闪切
- [x] 支持键盘方向键 / 鼠标滚轮 / 点击切换
- [x] 文字说明不喧宾夺主

---

### P0-4 照片懒加载 · 性能保障

**功能描述**
- 页面滚动到可视区域时才加载图片
- 加载中显示占位骨架或模糊预览
- 避免首屏等待时间过长

**验收标准**
- [x] 首屏加载时间合理（< 3s）
- [x] 滚动流畅，不卡顿
- [x] 图片加载失败有兜底占位图

---

### P0-5 响应式布局 · PC 优先

**功能描述**
- 以 PC 宽屏为主要设计场景
- 基本适配常见浏览器窗口大小
- 不破坏视觉体验

**验收标准**
- [x] 1280px 以上宽度体验完整
- [x] 窗口缩放时布局不错乱
- [x] 移动端留到后续迭代（P2）

---

## P1 · 有时间再做（加分项）

### P1-1 关于页（About）

- 摄影师简介 + 联系方式
- 1～2 张代表照片
- 简单即可，不要做成长文

### P1-2 照片全屏放大查看

- 点击照片进入全屏模式
- 支持放大 / 缩小 / 拖拽查看细节
- Esc 退出全屏

### P1-3 页面切换过渡动画

- 系列之间跳转有整体过渡动效
- 统一视觉风格，体现细节质感

### P1-4 深色 / 浅色模式切换

- 默认深色（展厅感）
- 可选切换到浅色（白盒美术馆感）

---

## P2 · 后续迭代（MVP 阶段不做）

- 移动端 / 小程序适配
- 照片上传后台 / CMS 管理
- 访客统计
- 评论 / 留言功能
- 多语言支持
- SEO 优化
- 自定义域名配置引导

---

## 内容规划（MVP 阶段）

| 系列 | 照片数量 | 说明 |
|------|----------|------|
| 系列 1 | 8～15 张 | 风格最强的主打系列 |
| 系列 2 | 8～15 张 | 主题差异化的第二系列 |
| 系列 3 | 8～15 张 | 补充视角的第三系列 |

> 照片质量 > 照片数量，每张都应值得被看到。
