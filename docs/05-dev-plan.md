# 05 · 开发计划 · 里程碑

---

## 开发顺序原则

```
1. 先用硬编码静态数据把页面结构跑通
2. 视觉和交互体验打磨到位
3. 最后补全内容（照片替换、文案完善）
```

---

## 里程碑

### M0 · 项目骨架（1～2小时）

- [ ] `npm create vite@latest` 初始化 Vue 3 项目
- [ ] 安装依赖：`vue-router` `pinia` `tailwindcss` `gsap` `@vueuse/core`
- [ ] 创建页面空组件：`HomePage` `CollectionPage` `AboutPage`
- [ ] 配置路由（`/` `/collection/:id` `/about`）
- [ ] 创建 `src/data/collections.js` 静态数据文件
- [ ] 准备至少 2 个系列的照片素材，放入 `src/assets/photos/`

**验收**：三个页面能正常跳转，静态数据可被读取

---

### M1 · 首页（2～3小时）

- [ ] `HeroCover.vue`：全屏封面图 + 网站标题 + 进入按钮
- [ ] `CollectionGrid.vue`：系列卡片网格展示
- [ ] Hover 动效（封面图缩放）
- [ ] 点击系列卡片跳转到对应 CollectionPage

**验收**：首页视觉完整，点击可进入系列页

---

### M2 · 系列展览页（3～4小时）

- [ ] `PhotoGrid.vue`：系列内照片网格
- [ ] `LazyImage.vue`：懒加载图片组件
- [ ] `PhotoViewer.vue`：点击后全屏大图弹层
- [ ] `usePhotoNavigation.js`：键盘 ← → / Esc / 滚轮切换逻辑
- [ ] 大图弹层的淡入淡出动效
- [ ] 系列标题 + 简介展示

**验收**：能进入系列页，网格展示，点击可全屏浏览，键盘可切换

---

### M3 · 导航 & 页面过渡（1小时）

- [ ] 全局顶部导航栏
- [ ] 滚动时导航半透明效果
- [ ] `PageTransition.vue`：页面切换过渡动画
- [ ] 返回首页按钮

**验收**：页面间切换有过渡，导航功能正常

---

### M4 · 视觉打磨（2～3小时）

- [ ] 统一字体引入（Playfair Display + Noto Serif SC）
- [ ] 深色主题色彩体系完善
- [ ] 照片说明 caption 样式
- [ ] 各处间距、排版细节对齐
- [ ] 加载占位骨架屏样式

**验收**：整体视觉一致，有展览质感

---

### M5 · 内容填充 & 发布（1小时）

- [ ] 所有系列照片最终版本替换
- [ ] 所有文案（系列简介、图片说明）最终版本
- [ ] `vite build` 构建无报错
- [ ] 部署到 Vercel / Netlify / GitHub Pages

**验收**：线上可访问，内容完整，无明显 bug

---

### M6 · P1 功能（可选，有时间再做）

- [ ] `AboutPage.vue` 完善
- [ ] 深色 / 浅色模式切换
- [ ] 全屏放大 + 缩放查看照片细节

## 技术债务记录（后续处理）

| 技术债 | 原因 |
|--------|------|
| 照片资源直接放 src/assets | 后续迁移到 CDN 或对象存储 |
| 无移动端适配 | 后续迭代手机端 / 小程序 |
| collections.js 手动维护 | 后续可接入简单 CMS |
| 无访客统计 | 上线后接入 Google Analytics 或 umami |
