# Ruby Love — 个人摄影展示网站

> **一个沉浸式的个人摄影作品展示网站，把照片变成一场可浏览的数字展览。**

## 技术栈

- **前端**：Vue 3 + Vite + Pinia + Vue Router + Tailwind CSS + GSAP
- **UI 组件**：Inspira UI（Vue 原生）· 参考 Aceternity UI · 参考 Magic UI
- **后续扩展**：移动端 H5 / 小程序
